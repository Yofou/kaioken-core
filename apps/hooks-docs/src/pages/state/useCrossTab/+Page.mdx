# useCrossTab

The useCrossTab hook provides a synchronized state across multiple tabs (or iframes) of the same web application using the BroadcastChannel API. It works similarly to **useSignal**, but any changes to the state are broadcasted to all open tabs, ensuring consistent state updates across different browser contexts.

## Usage

```tsx
const count = useCrossTab("counter", 0)

count.value += 1
```
